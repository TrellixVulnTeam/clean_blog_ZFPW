# clean_blog
Simple blog developed by django



### Usage:

- 下载源代码

  ```
  git clone git@github.com:WilliamsZhang/clean_blog.git
  ```

- 安装依赖包

  ```
  pip install -r requestments.txt
  ```

- 创建管理员

  ```
  python manage.py createsuperuser
  ```



### Home address

```
http://127.0.0.1:8000/index/
```
